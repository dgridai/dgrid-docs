# 支援的國家與地區

_最後更新：4 月 22 日（週三）_

## 概述

本頁面說明 DGrid AI Gateway（以下稱「AI Gateway」）的區域可用性考量。

AI Gateway 可能無法於所有國家、領土或司法管轄區提供。其可用性可能因法律、監管、制裁、出口管制、基礎設施、付款、營運、安全或第三方供應商限制而有所不同。

本頁面僅適用於 AI Gateway，並不取代其他可能適用於 DGrid 其他產品之區域限制、法律通知或產品專屬條款。關於服務使用要求，請參閱[服務條款](/zh-TW/AI-Gateway-Terms-of-Service)。

## 受限制或不支援的地區

若 DGrid 認定於某一國家、領土或司法管轄區提供 AI Gateway 存取或使用將帶來法律、監管、制裁、出口管制、安全、營運或商業風險，DGrid 得限制或拒絕提供該等存取。

在部分情況下，DGrid 可能：

- 阻止帳戶建立或 API 存取；
- 限制特定模型或功能的使用；
- 要求額外驗證或審查；或
- 如繼續提供服務已不被允許或不可行，則暫停或終止存取。

使用者不得自受限制或不支援地區，以規避地理、技術、合規或契約控制之方式存取或使用 AI Gateway。

## 各地區的服務限制

即使 AI Gateway 在某地區可用，並不代表所有功能、模型、端點、整合或帳戶選項都會在該地區提供。

地區差異可能源於：

- 第三方模型或供應商限制；
- 當地合規要求；
- 資料路由或基礎設施限制；
- 付款或計費能力；
- 語言或支援覆蓋範圍；或
- 營運上線時程。

DGrid 保留隨時調整、限制或撤回任何 AI Gateway 功能區域可用性的權利。

## 合規與出口管制

AI Gateway 的使用受所有適用的貿易限制、制裁法律、出口管制規則及相關監管要求約束。

你有責任確保自己、你的組織、你的使用者及你的使用情境所涉司法管轄區內，對 AI Gateway 的存取與使用符合所有適用法律與限制。

在特定地區或特定使用情境下，DGrid 可能要求額外審查、文件、驗證或契約性保證，方會啟用或持續提供存取。

## 區域可用性的變更

在適用法律允許範圍內，DGrid 得隨時就任何地區更新、擴大、限制、暫停或終止 AI Gateway 的可用性，且得有或無事先通知。

造成變更的原因可能包括：

- 法律或法規之變動；
- 制裁或出口管制之發展；
- 第三方供應商支援情況之變更；
- 基礎設施、安全或營運問題；或
- DGrid 內部之合規或風險評估。

使用者應參考 DGrid 最新發布之文件，以取得目前區域可用性之指引。

# 學術研究支撐

關於 PoQ 機制的設計邏輯、技術可行性與性能優勢，已通過專門的學術研究論證，相關成果可參考以下論文詳細了解：

1. [《Design and Evaluation of Cost-Aware PoQ for Decentralized LLM Inference》](https://https//arxiv.org/abs/2512.16317)：提出成本感知型 PoQ 框架，將效率測量融入獎勵機制，通過統一評估流水線實現質量與成本的動態平衡，驗證了多維度評估體系的實用性與經濟可持續性。

2. [《Optimistic TEE-Rollups: A Hybrid Architecture for Scalable and Verifiable Generative AI Inference on Blockchain》](https://https//arxiv.org/abs/2512.20176)：提出的混合驗證協議，為鏈上可驗證證明提供了底層架構支撐，通過可信執行環境（TEE）與零知識點檢相結合的方式，解決了去中心化推理的可驗證性三元困境，進一步強化了 PoQ 證明的安全性與高效性。

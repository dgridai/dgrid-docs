# 什麽是 $DGAI？

**$DGAI** 是 DGrid.AI 生態系統的原生實用型代幣，作為我們去中心化人工智能推理網絡中價值交換、激勵對齊和網絡治理的核心支柱。該代幣專為無縫集成於 Web3 基礎設施而設計，可實現用戶、節點運營商和開發者之間的無信任交互，從而消除對中間商的依賴。

## 技術規格

- **標準**：BRC-20代幣，兼容BNB Chain及所有EVM兼容區塊鏈，確保與錢包、交易所及智能合約的廣泛互操作性。
- **總供應量**：固定為**10億**枚代幣，上線後不再進行通脹式鑄造。這種稀缺性確保長期價值穩定，並使激勵機製與網絡增長保持一致。
- **關鍵擴展**：集成EIP-2612（許可）以實現無gas費批準，簡化推理任務和質押操作的支付流程。

# 服務條款

_最後更新：4 月 22 日（週三）_

## 概述

本服務條款（以下稱「本條款」）規範你對 DGrid AI Gateway（以下稱「AI Gateway」）之存取與使用，包括 DGrid 提供的相關 API、文件、工具、介面及服務功能。

AI Gateway 是一項用於存取、路由、管理及整合 AI 模型能力與相關功能的服務。除非 DGrid 另以書面明示，否則本條款僅適用於 AI Gateway 及其相關服務功能。

## 本條款的適用範圍

本條款適用於你對 AI Gateway 作為一項服務之存取與使用。

為免疑義，除非 DGrid 另以書面明示，本條款不適用於 DGrid Premium 或任何與 DGrid Premium 許可證相關之產品或安排。若 DGrid 就其他產品或服務另行發布專屬條款、政策、通知或商務條件，則應以該等文件為準。

當你存取或使用 AI Gateway，即表示你同意遵守本條款，以及 DGrid 可能不時就 AI Gateway 發布的其他文件、政策、使用要求或技術指引。

## 使用資格與合規要求

你僅得於適用法律允許之情況下使用 AI Gateway，並應在適用時有權代表你本人或你所代表的法人實體行事。

你有責任確保自己對 AI Gateway 的存取與使用，符合與你的司法管轄區、產業及使用情境相關的一切適用法律、法規、制裁限制、出口管制要求及內部合規義務。

如 DGrid 合理認定相關存取或使用將構成法律、監管、安全或合規風險，DGrid 得限制、暫停或拒絕提供 AI Gateway 存取權限。

## 帳戶與 API 憑證

若要使用 AI Gateway 的部分功能，你可能需要建立帳戶、取得 API 憑證，或使用 DGrid 核發之憑證。

你應負責：

- 維護你的帳戶與 API 憑證之機密性與安全性；
- 承擔所有透過你的帳戶或憑證進行之活動；
- 確保憑證不會被分享、外洩、轉售或以未經授權之方式使用；以及
- 一旦懷疑發生未經授權之存取、憑證外洩或濫用情形，應立即通知 DGrid。

如因安全、營運、法律或合規目的而有合理必要，DGrid 得輪替、撤銷、暫停或更換相關憑證。

## 允許的使用方式

在遵守本條款的前提下，你僅得基於合法、經授權且有文件支持之目的使用 AI Gateway。

你應依照 DGrid 所發布的技術文件、使用限制、產品指引及服務專屬限制使用 AI Gateway。

## 禁止的使用方式

你不得將 AI Gateway 用於：

- 違反任何適用法律或法規；
- 侵害、挪用或以其他方式侵犯 DGrid 或任何第三方之權利；
- 提交、傳輸或處理非法、有害、欺騙性、濫用性、侵權或其他被禁止之內容；
- 試圖未經授權存取任何系統、帳戶、模型、資料或網路；
- 干擾、破壞、降低或規避 AI Gateway 或相關系統之安全性、完整性或可用性；
- 反向工程、擷取、抓取、基準測試或複製本服務，但適用法律明確允許者不在此限；
- 規避速率限制、存取控制、安全保護、地理限制或使用限制；
- 以可能對 DGrid、其供應商或其他使用者造成不合理安全、法律、聲譽或營運風險之方式使用 AI Gateway；
- 除非 DGrid 以書面明確允許，否則轉售、再授權或向第三方提供未經授權的 AI Gateway 存取；或
- 將 AI Gateway 用於 DGrid 文件、政策頁面或適用服務通知中列明之任何禁止或受限制使用情境。

DGrid 得不時更新或補充 AI Gateway 的禁止使用限制。

你應就客戶內容自行負全部責任，包括：

- 其合法性、正確性、品質與適切性；
- 取得一切必要之權利、同意、通知與許可；以及
- 確保透過 AI Gateway 提交及處理客戶內容，不會違反法律、契約、保密義務或第三方權利。

在遵守適用法律、技術架構、已發布文件及 DGrid 就特定服務所作承諾之前提下，DGrid 得於合理必要範圍內處理客戶內容，以提供、維護、保護、支援及改善 AI Gateway。

DGrid 不對使用者提交之客戶內容承擔責任。

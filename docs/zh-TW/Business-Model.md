# 商業模式：三層收入模型

DGrid 的收入直接源於並反哺於網絡的價值創造環節，實現生態可持續發展：

## 1. 核心層：協議交易手續費

- 網絡交易抽成（2–10%）：對透過 DGrid 網絡完成的所有 AI 推理與 Agent 服務請求收取基礎費用，構成持續現金流核心。
- 市場服務費：對模型與 Agent 在市場中的上架、推廣及促成交易收取相應服務費用。

## 2. 擴展層：企業級與增值服務

- 企業訂閱（SLA）：為對可靠性、隱私有更高要求的企業客戶，提供高可用性保障、私有化部署及專屬支持服務。
- 高級 API 服務：提供更高調用频次、專屬模型/Agent 訪問權限、數據分析儀表盤等增值功能。

## 3. 生態層：資產化與金融化服務

- 質押服務收益：從節點及 AI 服務提供者的質押體系中獲取部分收益。
- 資產發行與交易費：為優質模型與 Agent 的通證化（Tokenization）發行、交易提供平台，並收取相關費用，捕獲生態增長的上層價值。

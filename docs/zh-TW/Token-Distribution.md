# 代幣分配

![Token Distribution](/Token_Distribution.png)


$DGAI 的總供應量固定為 10 億枚代幣，其分配設計旨在協調用戶、節點運營商、開發者及更廣泛生態系統之間的激勵機製。代幣通過結構化渠道進入流通，優先考慮長期網絡增長而非短期投機。10 億枚 $DGAI 代幣的分配如下：

- **節點 (50%)**
- **社區 (20%)**
- **團隊激勵 (10%)**
- **投資人 (10%)**
- **空投 (5%)**
- **初始流動性 (5%)**

## 生態正向循環

更多模型&Agent 入駐 → 帶來更多選擇與更優價格 → 提升支付與質押需求 → 加快通證流通速度並增強稀缺性 → 強化價值支撐 → 吸引更多優質參與者 → 推動網絡持續增長。

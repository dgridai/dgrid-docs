# 代幣分配

![Token Distribution](/Token_Distribution.png)


$DGAI 的總供應量固定為 10 億枚代幣，其分配設計旨在協調用戶、節點運營商、開發者及更廣泛生態系統之間的激勵機製。代幣通過結構化渠道進入流通，優先考慮長期網絡增長而非短期投機。10 億枚 $DGAI 代幣的分配如下：

- **節點 (50%)**：透過激勵創世高級會員(Genesis Premium)、智慧網關(Smart Gateway)及大型語言模型供應商(LLM Provider)，我們維持去中心化推理網絡的安全性與穩定性。此機制採用十年挖礦週期，每兩年進行一次挖礦獎勵減半事件。
- **社區 (20%)**：用於推動開發者資助、戰略合作、跨鏈集成及社區倡議，TGE（代幣生成事件）當下解鎖 30%，用於預挖代幣兌換與 TGE 運營活動；剩餘採 2 年線性解鎖。
- **團隊激勵 (10%)**：補償創始工程師與核心貢獻者，確保團隊長期投入，採 2 年線性解鎖機制。
- **投資人 (10%)**：分配給支持主網前開發工作的戰略投資人，TGE 當下解鎖 10%，剩餘採 2 年線性解鎖。
- **空投 (5%)**：交易所上幣相關激勵，TGE 當下 100% 解鎖。
- **初始流動性 (5%)**：預留用於中心化交易所（CEX）與去中心化交易所（DEX）的市場做市，TGE 當下 100% 解鎖。

## 生態正向循環

更多模型&Agent 入駐 → 帶來更多選擇與更優價格 → 提升支付與質押需求 → 加快通證流通速度並增強稀缺性 → 強化價值支撐 → 吸引更多優質參與者 → 推動網絡持續增長。

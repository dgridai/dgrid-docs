# 토큰 분배

![Token Distribution](/Token_Distribution.png)

$DGAI 의 총 공급량은 10억 개의 토큰으로 고정되어 있으며, 그 분배 설계는 사용자、노드 운영자、개발자 및 더 넓은 생태계 간의 인센티브 메커니즘을 조정하는 것을 목표로 합니다. 토큰은 구조화된 채널을 통해 유통에 진입하며, 단기投기보다 장기적인 네트워크 성장을 우선시킵니다. 10억 개의 $DGAI 토큰 분배는 다음과 같습니다:

- **노드 (50%)**
- **커뮤니티 (20%)**
- **팀 인센티브 (10%)**
- **투자자 (10%)**
- **에어드롭 (5%)**
- **초기 유동성 (5%)**

## 긍정적 생태계 순환

더 많은 모델 및 Agent 입주 → 더 많은 선택과 더 우수한 가격 제공 → 결제 및 스테이킹 수요 증가 → 토큰 유통 속도 가속화 및 희소성 강화 → 가치 지원 강화 → 더 많은 고품질 참여자 유치 → 네트워크 지속적 성장 추진.
